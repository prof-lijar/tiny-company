# Marketing Roadmap — Tiny Company

This document outlines the strategic marketing sequence for Tiny Company. Since the product is currently being defined by the CEO and Product Lead, this roadmap serves as the execution framework to be triggered upon product finalization.

## Phase 1: Foundation (Post-Product Definition)
*Goal: Establish the core messaging and target audience.*
- [ ] **Target Audience Analysis**: Define the Ideal Customer Profile (ICP) based on the product spec.
- [ ] **Value Proposition**: Develop the "Why us?" and "So what?" statements.
- [ ] **Tagline Exploration**: Generate and select a compelling company tagline.
- [ ] **Messaging House**: Create a framework of core pillars and supporting claims.

## Phase 2: Asset Creation
*Goal: Build the materials needed for a successful launch.*
- [ ] **Marketing Plan**: Document the channels, budget (if any), and growth strategy.
- [ ] **Landing Page Copy**: High-conversion copy for the home page.
- [ ] **Press Release**: Draft the official announcement.
- [ ] **Social Media Kit**: Templates and a distribution plan for X, LinkedIn, etc.

## Phase 3: Launch Execution
*Goal: Generate initial traction and user acquisition.*
- [ ] **Launch Checklist**: Final pre-flight check of all assets.
- [ ] **Distribution**: Execute the social media and press outreach.
- [ ] **Feedback Loop**: Monitor early user reactions to refine messaging.

## Phase 4: Growth & Optimization
*Goal: Scale the user base and optimize conversion.*
- [ ] **Content Strategy**: Plan for blog posts, case studies, and tutorials.
- [ ] **Referral Program**: Design a growth loop to encourage user invites.
- [ ] **KPI Tracking**: Monitor CAC, LTV, and conversion rates.
