# TOPIK Learning Assistant — Product Roadmap

This roadmap outlines the strategic milestones for the TOPIK Learning Assistant, moving from a Minimum Viable Product (MVP) to a comprehensive, AI-powered learning ecosystem.

## Phase 1: The Foundation (MVP)
**Goal**: Establish the core study loop and provide immediate value through high-quality practice materials.

### Milestone 1.1: Core Study Infrastructure
- [ ] **Landing Page**: High-conversion page with value prop and pricing.
- [ ] **Vocabulary Builder**: Implementation of SRS flashcards for Levels 3-6.
- [ ] **Grammar Library**: Level-organized lessons with examples and usage notes.
- [ ] **Reading Practice**: TOPIK-style passages with multiple-choice questions and explanations.

### Milestone 1.2: Basic User Experience
- [ ] **Korean Typography**: Integration of Noto Sans KR for optimal readability.
- [ ] **Responsive Layout**: Desktop-first study interface that works on mobile.
- [ ] **Content Seed**: Initial set of original TOPIK-style content for all P0 features.

---

## Phase 2: The Differentiators (Fast Follow)
**Goal**: Implement the "killer features" that separate the platform from mobile-only competitors.

### Milestone 2.1: AI Writing Lab
- [ ] **Writing Prompts**: Implementation of Tasks 51, 52, 53, and 54.
- [ ] **AI Feedback Engine**: Integration of LLM for grammar correction and scoring.
- [ ] **Character Counter**: Real-time TOPIK-specific character tracking.

### Milestone 2.2: IBT Simulation
- [ ] **Mock Test Engine**: Timed simulation of a full TOPIK II exam.
- [ ] **IBT Interface**: UI that mimics the actual computer-based testing environment.
- [ ] **Listening Module**: Audio-integrated practice and mock test sections.

---

## Phase 3: Personalized Learning (Growth)
**Goal**: Increase user retention and LTV through personalization and progress tracking.

### Milestone 3.1: User Ecosystem
- [ ] **Authentication**: Secure sign-up/sign-in and profile management.
- [ ] **Progress Dashboard**: Visual tracking of vocab mastery, grammar completion, and test scores.
- [ ] **Study Streaks**: Gamification elements to encourage daily habit formation.

### Milestone 3.2: Monetization & Scaling
- [ ] **Subscription Tiers**: Implementation of Free vs Pro tiers via Stripe.
- [ ] **Pro Paywall**: Locking AI feedback and full mock tests behind the subscription.
- [ ] **Advanced Analytics**: Identification of "weak areas" based on user error patterns.

---
