# TOPIK Learning Assistant \u2014 Product Roadmap

This roadmap outlines the strategic milestones for the TOPIK Learning Assistant, moving from a Minimum Viable Product (MVP) to a comprehensive, AI-powered learning ecosystem.

## Phase 1: The Foundation (MVP) - [COMPLETED]
**Goal**: Establish the core study loop.
- [x] Landing Page
- [x] Vocabulary Builder (SRS)
- [x] Grammar Library
- [x] Reading Practice
- [x] Korean Typography & Responsive Layout

---

## Phase 2: The Differentiators (Fast Follow) - [COMPLETED]
**Goal**: Implement \"killer features\" for web-based IBT.
- [x] AI Writing Lab
- [x] IBT Simulation
- [x] Listening Module

---

## Phase 3: Personalized Learning & Growth - [COMPLETED]
**Goal**: Increase user retention and LTV.
- [x] User Ecosystem (Auth, Dashboard, Streaks)
- [x] Monetization (Stripe, Subscription Tiers)
- [x] Quality Assurance & Polish (Ongoing)

---

## Phase 4: 2026 Format Alignment - [COMPLETED]
**Goal**: Pivot the entire product to align with the 2026 TOPIK Reform.
- [x] **Mock Test Overhaul**: Update question counts (Listening 60, Reading 40) and timers.
- [x] **High-Speed Listening**: Implement actual 1.1x audio playback speed.
- [x] **New Scoring Logic**: Implement 2026 thresholds (Level 4: 140, Level 5: 170).
- [x] **Anti-Template Writing AI**: Update LLM prompts to penalize formulaic writing.
- [x] **Reading Speed Training**: Adjust practice timings for 2026 constraints.

---
