# Tech Stack

The TOPIK Learning Assistant is built using a modern web stack to ensure responsiveness, type safety, and fast deployment.

## Frontend
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Hooks / Context API

## Backend & Database
- **API**: Next.js Route Handlers (Edge/Node.js runtime)
- **Database**: Supabase (PostgreSQL)
- **Authentication**: NextAuth.js / Supabase Auth

## Infrastructure
- **Deployment**: Vercel
- **CI/CD**: GitHub Actions

## Key Libraries
- **Audio**: Web Audio API (for pronunciation and listening)
- **PDF**: Embedded PDF viewer for mock tests
- **AI**: OpenAI/Anthropic API (for writing feedback and weakness analysis)
