# Demo Script: The 'Fix-It' Button in Action
**Objective:** Provide a high-impact, 45-60 second visual proof of the "Correction Gap" being closed.
**Target Audience:** Framework Partners (e.g., CrewAI), Technical Leads.
**Key Message:** "Don't just detect the loop; apply the surgical fix."

---

## Scene 1: The Pain (The Loop)
**Visual:** 
- Screen recording of a CLI or a basic UI showing an agent in a reasoning loop.
- The agent is repeating the same search query over and over.
- A red highlight appears over the repeating lines.
**Audio/Overlay:** 
- Text overlay: "The Reasoning Loop: The Silent Killer of Reliability."
- Sound: Subtle, repetitive ticking clock.

## Scene 2: The Detection (The Mirror)
**Visual:** 
- Transition to the TraceWhisper v2.1 view.
- The raw logs are synthesized into a narrative.
- A clear alert pops up: `[Reasoning Loop Detected]`.
- The "Divergence Point" is highlighted in yellow.
**Audio/Overlay:** 
- Text overlay: "Detection is only 50% of the solution."
- Sound: A "ping" indicating detection.

## Scene 3: The Magic (The Surgeon)
**Visual:** 
- The cursor moves to a prominent, glowing **"Fix-It"** button.
- Click!
- A side-by-side view appears: 
    - **Left:** The current failing prompt.
    - **Right:** The suggested "Surgical Adjustment" (highlighted in green).
- The suggestion is specific: *"Before searching, verify if the query was executed in the last 3 steps. If so, pivot to [X]."*
**Audio/Overlay:** 
- Text overlay: "Stop Guessing. Start Fixing."
- Sound: A satisfying "shink" or "unlock" sound.

## Scene 4: The Result (The Resolution)
**Visual:** 
- The suggested prompt is applied.
- The agent is run again.
- This time, the agent bypasses the loop, reaches the goal, and outputs the final answer.
- A green checkmark appears: `[Loop Broken]`.
**Audio/Overlay:** 
- Text overlay: "Close the Correction Gap. Welcome to the Reasoning IDE."
- Sound: Uplifting, fast-paced synth music swells.

## Scene 5: Call to Action
**Visual:** 
- Logo: TraceWhisper v2.2
- Text: "Join the Early Access List."
- URL: tracewhisper.ai/v2-early-access
**Audio/Overlay:** 
- Sound: Music fades out.
